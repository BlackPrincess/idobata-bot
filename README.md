### Testing Hubot Locally

You can test your hubot by running the following.

```
export HUBOT_IDOBATA_API_TOKEN=<token>
export HUBOT_NAME=<botname>
bin/hubot -a idobata
```

see [idobata/hubot-idobata](https://github.com/idobata/hubot-idobata)
